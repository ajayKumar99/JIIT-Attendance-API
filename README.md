# Usage

The API works by posting the below JSON structure.

```
{
  "eno" : "YOUR ENROLLMENT NO",
  "dob" : "DATE OF BIRTH",
  "password" : "WEBKIOSK PASSWORD"
}

```

The following JSON response is received on successful access.

```
{
  "attendance" : [
                  YOUR ATTENDANCE DETAILS
                  ]
}

```
